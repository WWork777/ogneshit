# Инструкция по добавлению изображений

Для полноценной работы лендинга необходимо добавить следующие изображения в папку `public/`:

## Hero секция

- `/hero-airport.jpg` - фоновое изображение для главной секции (аэропорт)
- `/hero-industry.jpg` - фоновое изображение (промышленность) - опционально
- `/hero-energy.jpg` - фоновое изображение (энергетика) - опционально

## About секция

- `/about-building.jpg` - фоновое изображение здания для секции "О компании"

## Projects секция

- `/project-expo.jpg` - проект EXPO 2017
- `/project-hotel.jpg` - проект Vladivostok Grand Hotel & Spa
- `/project-tennis.jpg` - теннисный центр в Лужниках
- `/project-airport.jpg` - проект аэропорта

## Directions секция

- `/direction-glass.jpg` - светопрозрачные конструкции
- `/direction-facade.jpg` - навесные вентилируемые фасады
- `/direction-research.jpg` - научная деятельность
- `/direction-restoration.jpg` - реставрация витражных конструкций
- `/direction-design.jpg` - проектирование фасадов
- `/direction-installation.jpg` - монтаж

## Product Catalog секция

- `/product-door1.jpg` - противопожарные двери -IW-50
- `/product-door2.jpg` - противопожарные двери -IW-60
- `/product-door3.jpg` - противопожарные двери -IW-90

## Рекомендации по размерам изображений:

- Hero изображения: 1920x1080px или больше
- Project/Direction/Product карточки: 800x600px или больше
- About background: 1200x800px или больше

Все изображения должны быть оптимизированы для веба (WebP, JPEG с качеством 80-85%).
