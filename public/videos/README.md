# Видео файлы

Здесь должны храниться локальные видео файлы для страницы проектов.

## Формат файлов

- **Рекомендуемый формат**: `.mp4` (H.264 codec)
- **Альтернативные форматы**: `.webm`, `.ogg`
- **Рекомендуемое разрешение**: 1920x1080 (Full HD) или выше
- **Рекомендуемый битрейт**: 5-10 Мбит/с для хорошего качества

## Именование файлов

Используйте понятные имена файлов, соответствующие названию проекта:

- `expo-2017.mp4`
- `vladivostok-hotel.mp4`
- `luzhniki-tennis.mp4`
- и т.д.

## Настройка видео

Убедитесь, что файлы видео оптимизированы для веб-использования:

- Используйте сжатие для уменьшения размера файла
- Рекомендуемый размер файла: до 100-200 МБ на видео
- Для длинных видео (более 5 минут) рекомендуется использовать более высокое сжатие

## Добавление нового видео

1. Загрузите видео файл в эту папку
2. Откройте файл `src/data/videos.js`
3. Добавьте новую запись в массив `videosData`:

```javascript
{
  id: 7,
  title: 'Название вашего видео',
  videoUrl: '/videos/имя-файла.mp4',
  thumbnail: '/images/Projects/изображение-превью.webp',
}
```
